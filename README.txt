---------------O   Algoritmos dividir y conquistar   O----------------

Códigos para benchmark de diferentes algoritmos

-----------------O   Algoritmos de ordenamiento   O-------------------

Se incluyen los siguientes algoritmos de ordenamiento:

1. Selection Sort (en "selectionSort.cpp")
2. Quick Sort (en "quickSort.cpp")
3. Merge Sort (en "mergeSort.cpp")
4. Sort Estándar de C++ (en "stdSort.cpp")

-----------O   Algoritmos de multiplicación de matrices   O-----------

También están los siguientes algoritmos de multiplicación de matrices

1. Iterativo cúbico tradicional (en "matrixNormal.cpp")
2. Iterativo cúbico optimizado (en "matrixTranspuesta.cpp")
3. Algoritmo de Strassen (en "matrixStrassen.cpp")

-------------------------O   Datasets   O-----------------------------

Para poder hacer los benchmark correspondientes estan los siguientes
códigos en python:

1. Datasets de ordenamiento (en "sortDatasets.py" y generados en
"datasetSort.txt")
2. Datasets de multiplicación de matrices (en "matrixDatasets.py" y
generados en "datasetMatrix.txt")

------------------------O   Compilación   O----------------------------

Para compilar los algoritmos necesitará Ubuntu y el compilador g++.
Abra la consola y escriba:

g++ -o nombre_compilado nombre_a_compilar
